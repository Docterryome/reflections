When would you want to use a remote repository rather than keeping all your work
local?

    If you have multiple people working on the same project, it's a lot easier to manage to get the work from a central place.

automatically stay up-to-date with your remote repository?

    You might be working on something really big that does not need to go to the repository right away, and somebody could have made a change on the file that you are working on and pushed it to the repository which might interfeare with what you are trying to do.
